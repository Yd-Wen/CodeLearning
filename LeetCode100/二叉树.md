## 二叉树

每个节点最多只有两个子节点（左孩子、右孩子）。

**结构**：
- 根节点：最顶上那个节点
- 左子树：左边的分支
- 右子树：右边的分支
- 叶子节点：最底下没有孩子的节点

**例子**：
```plaintext
            5      ← 根
        /   \
    3     7   ← 左孩子、右孩子
    / \   / \
2   4 6   8 ← 叶子
```

## 二叉树遍历

把树上所有节点按某种顺序 “走一遍、读一遍”。

一共有 **4 种常用遍历**：

- **前序遍历**：根节点 → 左子树 → 右子树（5-3-2-4-7-6-8）

- **中序遍历**：左子树 → 根节点 → 右子树（2-3-4-5-6-7-8）**二叉搜索树（BST）的中序遍历结果一定是升序数组！**

- **后序遍历**：左子树 → 右子树 → 根节点（2-4-3-6-8-7-5）

- **层序遍历**：逐层遍历，从上到下，从左到右（5-3-7-2-4-6-8）

> - 前序、中序、后序：深度优先搜索（DFS），栈 / 递归
> - 层序遍历：广度优先搜索（BFS），队列

Morris 算法实现中序遍历：

- 如果 x 无左孩子，先将 x 的值加入答案数组，再访问 x 的右孩子，即 x=x.right。
- 如果 x 有左孩子，则找到 x 左子树上最右的节点（即左子树中序遍历的最后一个节点，x 在中序遍历中的前驱节点），我们记为 predecessor。根据 predecessor 的右孩子是否为空，进行如下操作。
    - 如果 predecessor 的右孩子为空，则将其右孩子指向 x，然后访问 x 的左孩子，即 x=x.left。
    - 如果 predecessor 的右孩子不为空，则此时其右孩子指向 x，说明我们已经遍历完 x 的左子树，我们将 predecessor 的右孩子置空，将 x 的值加入答案数组，然后访问 x 的右孩子，即 x=x.right。

总结：
- 无左往右走（当前节点加入答案数组）；
- 有左找前驱；
- 前驱有右（当前节点）置为空，往右走（当前节点加入答案数组）。
- 前驱无右（当前节点）指当前，往左走。
> 前驱：左子树最右的节点。
> ```python
> prev = current.left
> while prev.right and prev.right != current: 
> prev = prev.right
> ```

## 二叉树构造

遍历序列组合	能否唯一构造二叉树？
前序 + 中序	✅ 能
后序 + 中序	✅ 能
前序 + 后序	❌ 不能（除了满二叉树，普通二叉树无法唯一确定）

> 为什么 前 + 后 不行？
> 因为无法区分左右子树范围，存在多棵不同树拥有相同前序、后序序列，所以无法唯一构造。

**例子**：前序 + 中序
- 前序遍历：根 → 左 → 右
→ 第一个元素一定是整棵树的根！
- 中序遍历：左 → 根 → 右
→ 找到根后，左边是左子树，右边是右子树！

**步骤**：
- 前序第一个 = 根
- 中序找到根，左边左子树，右边右子树
- 计算左子树大小
- 递归构造左、右子树

**总结**：前（后）序定位根，中序分左右

## 二叉搜索树

二叉搜索树（BST）：左小、根中、右大，且左右子树也都是二叉搜索树。

- 左子树上所有节点的值 < 根节点的值
- 右子树上所有节点的值 > 根节点的值
- 左、右子树也必须是 二叉搜索树

> 左小右大，中序递增
> 没有等于！BST 里一般没有重复值。

## 平衡二叉树

左右子树的高度差不超过 1。
