## 并查集

### 一、基本概念

并查集（Union-Find Set / Disjoint Set）是一种处理**元素分组/动态连通性**问题的数据结构，核心解决两个问题：
1. **查询**两个元素是否属于同一集合
2. **合并**两个集合

> 应用场景：判断图是否有环、网络连通性、Kruskal最小生成树算法等

### 二、三大核心操作

| 操作 | 作用 | 类比（公司找领导） |
|------|------|-------------------|
| **Make-Set** | 为每个元素创建独立集合，父节点指向自己 | 新员工入职，自己就是自己的领导 |
| **Find** | 查找元素所在集合的根节点（代表） | 一级级往上问，直到找到顶头上司 |
| **Union** | 合并两个不同集合 | 公司重组，两个部门合并，选一个老大 |

### 三、数据结构实现

**数组表示法**：`parent[i]` 表示元素 `i` 的父节点
- 下标 → 元素编号
- 值 → 父节点编号
- 根节点特征：`parent[root] == root`

```
视觉结构：多棵倒挂的树组成的森林
        0          4
       / \        / \
      1   2      5   6
         /          /
        3          7
```

### 四、版本演进与优化

#### 1.0：基础实现

**问题**：合并不当会导致树退化成链表，Find 最坏 O(n)

```python
class UnionFind:
    def __init__(self, n):
        self.parent = list(range(n))  # Make-Set：每个元素自成集合
    
    def find(self, x):
        # 一直找父节点，直到找到根
        while self.parent[x] != x:
            x = self.parent[x]
        return x
    
    def union(self, x, y):
        root_x = self.find(x)
        root_y = self.find(y)
        if root_x != root_y:
            self.parent[root_y] = root_x  # y的根挂到x的根下
```

#### 2.0：按秩合并（Union by Rank）

**优化思想**：合并时，矮树挂到高树下面，控制树高

```python
class UnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n  # 记录每棵树的高度（秩）
    
    def find(self, x):
        while self.parent[x] != x:
            x = self.parent[x]
        return x
    
    def union(self, x, y):
        root_x = self.find(x)
        root_y = self.find(y)
        if root_x == root_y:
            return
        
        # 矮树挂高树：保持树尽可能矮
        if self.rank[root_x] < self.rank[root_y]:
            self.parent[root_x] = root_y
        elif self.rank[root_x] > self.rank[root_y]:
            self.parent[root_y] = root_x
        else:
            self.parent[root_y] = root_x
            self.rank[root_x] += 1  # 高度相同，合并后高度+1
```

#### 2.1：按大小合并（Union by Size）

**优化思想**：小树挂到大树下面，用节点数量判断

```python
class UnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.size = [1] * n  # 记录每棵树的节点数
    
    def find(self, x):
        while self.parent[x] != x:
            x = self.parent[x]
        return x
    
    def union(self, x, y):
        root_x = self.find(x)
        root_y = self.find(y)
        if root_x == root_y:
            return
        
        # 小树挂大树
        if self.size[root_x] < self.size[root_y]:
            self.parent[root_x] = root_y
            self.size[root_y] += self.size[root_x]
        else:
            self.parent[root_y] = root_x
            self.size[root_x] += self.size[root_y]
```

#### 3.0：路径压缩 + 按秩合并（最终版）

**路径压缩**：Find 时把路径上所有节点直接挂到根节点，树变扁平

```python
class UnionFind:
    def __init__(self, n):
        self.parent = list(range(n))
        self.rank = [0] * n
    
    def find(self, x):
        # 递归路径压缩：找到根后，沿途节点全部直连根
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])  # 压缩！
        return self.parent[x]
    
    def union(self, x, y):
        root_x = self.find(x)
        root_y = self.find(y)
        if root_x == root_y:
            return
        
        # 按秩合并
        if self.rank[root_x] < self.rank[root_y]:
            self.parent[root_x] = root_y
        elif self.rank[root_x] > self.rank[root_y]:
            self.parent[root_y] = root_x
        else:
            self.parent[root_y] = root_x
            self.rank[root_x] += 1
    
    # 简化写法
    def union(self, x, y):
        x = self.find(x)
        y = self.find(y)
        if x == y:
            return
        # 高秩在前
        if self.rank[x] < self.rank[y]:
            x, y = y, x
        # 低秩并入
        self.parent[y] = x
        if self.rank[x] == self.rank[y]:
            self.rank[x] += 1
```

### 五、版本对比总结

| 版本 | 优化策略 | Find 复杂度 | Union 复杂度 | 特点 |
|------|----------|-------------|--------------|------|
| **V1.0** | 无 | O(n) 最坏 | O(n) | 简单直观，易退化 |
| **V2.0** | 按秩合并 | O(log n) | O(log n) | 控制树高 |
| **V2.1** | 按大小合并 | O(log n) | O(log n) | 控制树规模 |
| **V3.0** | 路径压缩 + 按秩合并 | **O(α(n)) ≈ O(1)** | **O(α(n)) ≈ O(1)** | 近乎常数，最优实用 |

> **α(n)** = 反阿克曼函数，在任意实际数据规模下值 ≤ 4~5，可视为常数

### 六、非整数元素处理

若元素是字符串等非整数类型，**用哈希表映射**：

```python
class UnionFindGeneric:
    def __init__(self):
        self.parent = {}   # key: 元素, value: 父节点
        self.rank = {}
    
    def add(self, x):
        if x not in self.parent:
            self.parent[x] = x
            self.rank[x] = 0
    
    def find(self, x):
        if self.parent[x] != x:
            self.parent[x] = self.find(self.parent[x])
        return self.parent[x]
    
    def union(self, x, y):
        self.add(x)
        self.add(y)
        root_x, root_y = self.find(x), self.find(y)
        if root_x == root_y:
            return
        if self.rank[root_x] < self.rank[root_y]:
            self.parent[root_x] = root_y
        elif self.rank[root_x] > self.rank[root_y]:
            self.parent[root_y] = root_x
        else:
            self.parent[root_y] = root_x
            self.rank[root_x] += 1
```
