# 链表（Linked List）

## 一、什么是链表？
链表是一种**线性数据结构**，和数组不同：
- 数组：内存**连续**，可以直接通过下标访问
- 链表：内存**不连续**，每个节点存 **值(val)** + **下一个节点的指针(next)**

标准**单链表**定义：
```python
class ListNode(object):
    def __init__(self, x):
        self.val = x    # 节点存储的值
        self.next = None # 指向下一个节点的指针（最后一个节点指向 None）
```

### 单链表结构示意图
```
头节点        节点2        节点3        尾节点
[1 | next] → [2 | next] → [3 | next] → None
```

### 链表核心特点（重要）
1. **不能随机访问**：想找第 k 个元素，必须从头遍历
2. **插入/删除效率高**：只需要改指针，不用移动元素（O(1)）
3. **空间灵活**：长度不固定，随用随分配

### 1.  dummyNode 哨兵节点（万能神器）
**解决问题**：头节点可能被修改（删除、反转、插入）时，避免特判头节点。
**用法**：创建一个虚拟节点，让它的 next 指向真实头节点。

#### 模板
```python
def function(head):
    dummy = ListNode(0)  # 创建哨兵
    dummy.next = head    # 指向原头节点
    cur = dummy          # 用 cur 遍历
    
    # 逻辑处理...
    
    return dummy.next    # 最终返回新头节点
```

**适用题目**：删除节点、合并链表、反转链表、两两交换节点等。

---

### 2. 双指针法（快慢指针 / 前后指针）
**解决问题**：
- 找链表中点
- 判断环
- 删除倒数第 N 个节点
- 找环入口

#### 模板 1：快慢指针（找中点 / 判断环）
```python
# 快慢指针初始化
slow = fast = head

# 找中点（偶数长度返回中间靠左）
def middle(head):
    slow = fast = head
    while fast.next and fast.next.next:
        slow = slow.next
        fast = fast.next.next
    return slow

# 判断环（相遇则有环）
while True:
    # 指向空节点，说明无环。
    if not fast or not fast.next:
        return None

    # fast 和 slow 异速前进
    fast = fast.next.next
    slow = slow.next
    if slow == fast:
        return True  # 有环
```

#### 模板 2：前后指针（删除倒数第 N 个）
```python
dummy = ListNode(0)
dummy.next = head
fast = slow = dummy

# fast 先走 N 步
for _ in range(n):
    fast = fast.next

# 一起走
while fast.next:
    fast = fast.next
    slow = slow.next

# slow.next 就是要删除的节点
slow.next = slow.next.next
```

### 3. 反转链表（高频必考）
LeetCode 最常考：**反转整个链表**、**反转区间链表**。

#### 模板：迭代反转（推荐，好写不易错）
```python
def reverse(head):
    pre = None
    cur = head
    while cur:
        next_node = cur.next  # 保存下一个节点
        cur.next = pre        # 反转
        pre = cur             # 移动 pre
        cur = next_node       # 移动 cur
    return pre  # pre 变成新头节点
```

**口诀**：**先存后，再反转，后移 pre 和 cur**。

---

### 4. 递归法（优雅但难理解）
**适用**：反转链表、回文链表、合并有序链表。

#### 模板：递归反转链表
```python
def reverse(head):
    if not head or not head.next:
        return head
    new_head = reverse(head.next)
    head.next.next = head
    head.next = None
    return new_head
```

---

### 5. 拆分与合并链表
**适用**：奇偶链表、分割链表、合并 K 个升序链表。

#### 模板：合并两个有序链表
```python
def mergeTwoLists(l1, l2):
    dummy = cur = ListNode(0)
    while l1 and l2:
        if l1.val < l2.val:
            cur.next = l1
            l1 = l1.next
        else:
            cur.next = l2
            l2 = l2.next
        cur = cur.next
    cur.next = l1 or l2  # 拼接剩余部分
    return dummy.next
```

## 三、LeetCode 链表题通用解题步骤（拿到题直接套）
1. **先看是否需要 dummy 节点**（头会变就用）
2. **确定用几个指针**（1 个遍历 / 2 个双指针）
3. **画图！** 指针怎么移动、怎么连接
4. **边界检查**：空链表、单个节点、最后一个节点
5. **返回正确的头节点**（dummy.next）

### 总结
1. **链表 = 值 + 指针，不连续存储**
