## 哈希表

1. 哈希表 = Python 字典 dict
2. 存的是 键: 值，查找速度极快（O (1)）
3. 两数之和里：
        - 键 = 数组数字
        - 值 = 数字下标
4. 核心逻辑：
**遍历一个数 → 查（目标 - 当前数）是否存在 → 存在就返回答案**

## 应用场景

一查二去三计数，四映五替六提速。

1. 查：查找是否存在 不用遍历列表一个个找 O (1) 瞬间出结果
2. 去：去重 存过的 key 不会重复 天然自带去重功能
3. 计数：统计频率 key = 元素 value = 出现次数
4. 映：键值映射
5. 替：代替数组下标 数字很大 / 负数 / 字符串，不能当数组下标
6. 提速：把 O (n²) → O (n) 凡是两层循环，优先想：能不能用哈希表干掉一层循环

## 常用方法
1. 创建空字典
```python
d = {}
d = dict()
```

2. 增加/修改键值对
```python
d[key] = value # 不存在就添加，存在就修改
```

3. 查找键是否在字典中
```python
if key in d: # 存在返回 True
if key in d.keys(): # 存在返回 True
if value in d.values(): # 存在返回 True
```

4. 获取值
```python
value = d[key] # 获取值
value = d.get(key) # 获取值
value = d.get("name", "默认值")  # 不存在就返回默认值
```

5. 删除键值对
```python
del d["name"]
d.pop("name")  # 也可以删除并返回值
```

6. 获取所有
```python
keys = d.keys()  # 返回所有键
values = d.values()  # 返回所有值
items = d.items()  # 返回所有键值对
for k, v in d.items(): # 遍历所有键值对
        print(k, v)
```

7. 清空字典
```python
d.clear()
d = {}
d = dict()
```

> 最常用的方法
> 1. 判断是否存在 
> ```python
> if key in d:
>   return True
> ```
> 2. 统计次数
> ```python
> count = d.get(num, 0)  # 不存在就是 0
> d[num] = count + 1
> ```
